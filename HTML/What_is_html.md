# HTML
---
> HTML이란 무엇인가?

**HyperText Markup Language**  

이름에 드러나듯, HTML은 웹페이지의 마크업에 사용되는 언어이다.  
상당히 모호한 이야기라고 느껴질 수도 있을 거 같다.

여하튼 HTML 문서 내용은 대략 이렇다.
1) 내용 기술
2) 뼈대 잡기
3) 여러 기능 - (이미지, 폼, 버튼, 링크, 시맨틱, 검색어 최적화 등등)
